# Detecao-de-Anomalias-em-Transacoes-em-Python
Detecção de fraudes em cartões de crédito com Python e Scikit-learn. Trata dados desbalanceados usando SMOTE, undersampling e class_weight. Compara Regressão Logística, Random Forest e XGBoost, avaliando com ROC/AUC, Precision-Recall e SHAP para explicabilidade do modelo.
